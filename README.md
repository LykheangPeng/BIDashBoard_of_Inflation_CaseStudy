# BIDashBoard_of_Inflation
This is a case study analysis project which using tools of Microsoft PowerBI under National Bank of Cambodia's Data Sheets.
Microsoft Power BI Dashboard of this project focused on Visualization of the Consumer Price Index, Core Inflation, Contribute Inflation Index of each categories items, and Inflation Rate.
